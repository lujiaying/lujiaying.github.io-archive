# lujiaying.github.io
My Blog

```branch source``` store source codes.

```branch master``` store deploy files.

## 操作指南
以下操作均在source分支下进行

新建一篇文章
`$ hexo new <title>`

生成静态页面
`$ hexo generate`

本地服务器预览,默认端口4000
`$ hexo server`

部署到服务器
`$hexo deploy`

## 配置指南
```
cd themes
git clone https://github.com/lujiaying/hexo-theme-yilia yilia
```
