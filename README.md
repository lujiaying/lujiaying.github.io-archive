## 操作指南 V1.0

使用Travis持续集成，只需在source分支下`$ hexo new <title>`，然后push代码即可自动构建。

---

## 操作指南 V0.5
以下操作均在source分支下进行

新建一篇文章
`$ hexo new <title>`

生成静态页面
`$ hexo generate`

本地服务器预览,默认端口4000
`$ hexo server`

部署到服务器
`$hexo deploy`

## 配置指南

npm install
cd themes
